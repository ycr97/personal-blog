# 查看日志常用命令

## grep

grep命令是个强大的文本搜索工具

grep命令的具体命令的语法格式为grep [参数] "关键字" [文件/目录]

参数

```shell
-i 搜索时，忽略大小写
-c 只输出匹配行的数量
-l 只列出符合匹配的文件名，不用列出具体的匹配行
-n 列出所有的匹配行，显示行号
-h 查询多文件时不用显示行号
-s 不显示不存在、没有匹配文本的错误信息
-v 显示不包含匹配文本的所有行
-w 匹配整词
-x 匹配整行
-r 递归搜索
```

## less

less命令可以更加随意的浏览文件，而且less在查看之前不会加载整个文件

使用方式: less [参数] 文件

```shell
-e 当文件显示结束后，自动离开
-f 强迫打开特殊文件，例如外围设备代号，目录和二进制文件
-g 只标志最后的搜索的关键词
-i 忽略搜索时的大小写
-m 显示类似more命令的百分比
-N 显示每行的行号
-Q 不使用警示音
-s 显示连续空行为一行
-S 行过长时间将超出部分舍弃
/字符串 向下搜索字符串
?字符串 向上搜索字符串
n 重复前一个搜索(与/或?有关)
N 反向重复前一个搜索(与/或?有关)
b 向前滚动半页
y 向前滚动一行
空格键 滚动一页
回车键 滚动一行
q 退出
[pagedown] 向下翻动一页
[pageup] 向上翻动一页
```

举例

1. 查看文件

   ```shell
   less log20220714.log
   ```

2. ps查看进程信息并通过less分页显示

   ```shell
   ps -ef|less
   ```

3. 查看命令历史使用记录并通过less分页显示

   ```shell
   history | less
   ```

4. 浏览多个文件

   ```shell
   less log1.log log2.log
   ```
