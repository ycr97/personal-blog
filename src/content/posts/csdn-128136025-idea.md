我们在日常开发中使用到最多的开发工具莫过于IDEA，这一款非常好用的Java集成开发环境。工具很强大有许多的使用技巧如更契合开发习惯的配置、IDEA常用的快捷键以及各式各样强大的插件，掌握这些技巧会使我们的工作效率得到极大提升，下面将介绍一些IDEA实用的使用技巧。

## 常用插件

#### 1. Translation

编码时经常会忘记该如何用英文给类、方法、变量等取名；或者阅读一些源码时注释是英文可以使用该插件来翻译

该插件的快捷键为Ctrl + Shift + O

![翻译](/personal-blog/images/csdn/128136025/01-b6872cb6e5a3.gif)

最近谷歌翻译的中文站点关闭可配置其他翻译引擎使其可用

![翻译配置](/personal-blog/images/csdn/128136025/02-38ee11118e71.png)

#### 2. MybatisX

该插件主要是可以实现mybatis Mapper与映射的XML之间的跳转；如动图演示：点击图标即可实现跳转；平时阅读代码非常方便。

![mybatisX.gif](/personal-blog/images/csdn/128136025/03-a918bd87bf50.gif)

#### 3. CodeGlance

CodeGlance会将当前Tab页的代码生成一个缩略的视图，对于一个类代码行数很多时使用该插件可以更方便的上下滑动，提高阅读代码的效率。

![CodeGlance.gif](/personal-blog/images/csdn/128136025/04-9f073d225e96.gif)

#### 4. GsonFormatPlus

项目开发过程中经常会对接三方接口，为了更方便的根据三方接口返回的JSON数据快速生成对应的实体类，GsonFormatPlus很好的解决这个问题；提升我们的开发效率

GsonFormatPlus快捷键: Alt + S

![GsonFormatPlus](/personal-blog/images/csdn/128136025/05-151af0031503.png)

#### 5. RestfulTool

1、根据 URL 直接跳转到对应的方法定义(快捷键 Ctrl + Alt + \\)
2、提供了一个 Services tree 的显示窗口
3、一个简单的 http 请求工具
4、直接导航到方法，复制接口地址，复制API接口，复制方法参数...
5、java 类上添加 Convert to JSON 功能，格式化 json 数据 

支持 Spring 体系 (Spring MVC / Spring Boot 1.x,2.x)
支持 JAX-RS
支持 Java 和 Kotlin 语言。

![RestfulTool](/personal-blog/images/csdn/128136025/06-924d79be11d8.png)

如上图所示，当我们选中url搜索该接口的方法定义快速定位到该方法

#### 6. Maven Helper

Maven Helper可以帮我们解决依赖冲突；在pom文件中下面有Dependency Analyzer的选项进入该选项进入下图的界面，插件会帮我们将有冲突的包列举出来；并且会将依赖以列表或树形的方式展示在界面上。使其更加直观和便捷地去解决依赖冲突和查看依赖。

![Maven Helper](/personal-blog/images/csdn/128136025/07-44d1f85e1d09.png)

#### 7. Alibaba Java Coding Guidelines

这款插件是阿里巴巴推出的代码规范扫描插件；在开发过程中开启实时扫描会在写代码时提示是否有不符合规范的代码。也可以针对单个文件或目录进行扫描，可以起到规范代码的作用并且会给出相应修改的提示；十分方便。

![Alibaba Java Coding Guidelines](/personal-blog/images/csdn/128136025/08-8eccaba7d24b.png)

#### 8. Lombok

使用Lombok插件的作用: 简化了Model层的代码的编写；以前pojo类/实体类，需要自己提供set、get、toString、equals、hashCode， Lombok通过各种注解，简化了以上操作。

安装好Lombok插件后还需要在我们的项目POM中Lombok的依赖

```xml
 <!--支持lombok-->
 <dependencies>
          <dependency>
               <groupId>org.projectlombok</groupId>
               <artifactId>lombok</artifactId>
               <version>1.18.24</version>
           </dependency>
 </dependencies>
```

注解:

- @Data：会自动生成set、get、toString、equals、hashCode
- @NoArgsConstructor：自动生成无参构造
- @AllArgsConstructor：自动生成全参构造
- @Accessors(chain = true)：开启链式编程
- @Builder: 开启链式编程给属性设置值

#### 9.Material Theme UI Plugin

这是一款可以为IDEA更换主题的插件；主题中有多种主题可供选择，根据自己的喜好来选择；可以根据自己的喜好来选择字体颜色、字体大小等设置。

主题设置: File > Settings > Appearance & Behavior > Material Theme > Settings

## 配置

#### 1. 代码提示不区分大小写

Settings -> Editor -> General -> Code Completion

#### 2. 多Tab页显示

Settings -> Editor -> General -> Editor Tabs

![多Tab页显示](/personal-blog/images/csdn/128136025/11-d132a4a69532.png)

#### 3. 方法分割线

Settings -> Editor -> General -> Appearance

![方法分割线.png](/personal-blog/images/csdn/128136025/12-def22315c8da.png)

#### 4. 代码编辑区显示行号

Settings -> Editor -> General -> Appearance

![显示代码行号.png](/personal-blog/images/csdn/128136025/13-b1ac553466d1.png)

#### 5.自动导包

Settings -> Editor -> General -> Auto Import 

![自动导包](/personal-blog/images/csdn/128136025/14-9226cf418107.png)

#### 6. 代码提示

Settings -> Keymap -> Main Menu -> Code

IDEA默认代码提示快捷键为Ctrl + 空格；但此快捷键易与输入法快捷键冲突；建议修改为自己习惯的快捷键。

![代码提示.png](/personal-blog/images/csdn/128136025/15-64d8dc75bd05.png)

#### 7. 滚动调整字体大小

Settings -> Editor -> General 

![鼠标滚轮修改代码.png](/personal-blog/images/csdn/128136025/16-a229d29841b6.png)

#### 8. 配置默认注释

我们编写代码时有些地方的代码是重复且必要的；比方说我们的类注释我们可以在设置中配置默认的注释模板在新建类时即可生成。

 Settings -> Editor -> File and Code Templates -> Files

```java
 /** 
  * @ClassName: ${NAME} 
  * @Description: TODO
  * @Author: ${USER}
  * @Date: ${DATE} ${TIME}
  * @Version: 1.0 
  **/
```

![注释模板](/personal-blog/images/csdn/128136025/17-f93ea150fd75.png)

Settings -> Editor -> General -> Appearance

##   快捷键

#### 1.查找

| 快捷键           | 介绍                     |
| ---------------- | ------------------------ |
| Ctrl + F         | 当前文件内查找文本       |
| Ctrl + R         | 当前文件内进行文本替换   |
| Ctrl + Shift + F | 在项目内进行全局文本查找 |
| Ctrl + Shift + R | 在项目内进行全局文本替换 |
| Shift + Shift    | 快速搜索                 |
| Ctrl + N         | 查找类                   |
| Ctrl + Shift +N  | 查找文件                 |

#### 2. 跳转切换

| 快捷键           | 介绍                   |
| ---------------- | ---------------------- |
| Ctrl + E         | 最近文件               |
| Ctrl + Tab       | 切换文件               |
| Ctrl + Alt + ←/→ | 跳转到历史停留处       |
| Alt+ ←/→         | 切换Tab                |
| Ctrl + G         | 跳转到当前文件指定行号 |

#### 3. 编码相关

| 快捷键           | 介绍                                                  |
| ---------------- | ----------------------------------------------------- |
| Ctrl + W         | 快速选中                                              |
| Ctrl + D         | 复制光标所在行或选中的代码块                          |
| Ctrl + Y         | 删除光标所在行或选中的代码块                          |
| Ctrl + Z         | 撤销                                                  |
| Ctrl + Shift + Z | 取消撤销                                              |
| Alt + Enter      | 智能提示                                              |
| Ctrl + Shift + L | 代码格式化                                            |
| Alt + Insert     | 代码生成，如生成对象的get/set、构造方法、toString()等 |
| Ctrl + Alt + M   | 提取方法                                              |
| Ctrl + Alt + O   | 优化import                                            |
| Ctrl + Alt + T   | 代码生成，如if、while、try/catch等代码块              |
| Alt + 鼠标左键   | 块选择                                                |
| Ctrl + /         | 单行注释                                              |
| Ctrl + Shift + / | 多行注释                                              |
| ...              | ...                                                   |

#### 4.  代码阅读相关

| 快捷键         | 介绍                 |
| -------------- | -------------------- |
| Ctrl + P       | 方法参数提示显示     |
| Alt + F7       | 查找变量被何处使用到 |
| Ctrl + H       | 类的层级关系         |
| Ctrl + Alt + B | 跳转方法实现         |

快捷键有很多就不一一例举了，以上介绍的快捷键都是我平时工作中经常用到一些快捷键；肯定还有些未用到的希望有所参考。
