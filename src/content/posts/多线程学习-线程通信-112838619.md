线程的通信案例:两个线程交替打印1-100的数

```java
/**
 * 线程通信例子两个线程打印1-100, 线程1,线程2交替打印
 * @author ycr
 * @date 2021/1/19
 */
public class CommunicationDemo {

    public static void main(String[] args) {
        Number number = new Number();

        Thread t1 = new Thread(number, "T1");
        Thread t2 = new Thread(number, "T2");

        t1.start();
        t2.start();
    }

}

class Number implements Runnable {

    private int num = 1;

    @Override
    public void run() {

        while (num <= 100) {
            synchronized (this) {
                // 唤醒阻塞状态的线程,调用者和上面的锁对象都是this
                notifyAll();
                if (num <= 100) {
                    try {
                        Thread.sleep(10);
                        System.out.println(Thread.currentThread().getName() + " : " + num);
                        num++;
                    } catch (InterruptedException e) {
                        e.printStackTrace();
                    }
                }
                try {
                    // 调用wait()方法使线程进入阻塞状态,调用者和上面的锁对象都是this
                    wait();
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }

            }
        }
    }
}

```
 **涉及到的三个方法:**
 1. wait():一旦执行此方法,当前线程进入阻塞状态,并释放同步监视器
 2. notify():一旦执行此方法,就会唤醒一个wait()的线程,如果有多个线程为阻塞状态则唤醒优先级最高的线程
3.  notifyAll():一旦执行此方法,就会唤醒所有wait()的方法

**说明:**
 1. wait(),notify(),notifyAll()三个方法必须使用在同步方法或同步代码块中
 2. wait(),notify(),notifyAll()三个方法的调用者必须是锁对象,如果使用的是不同的对象则会抛出IllegalMonitorStateException异常

**面试题:** sleep()和wait()异同
 1. 相同点: sleep()和wait()都可以使当前线程进入阻塞状态
 2. 不同点: 
 (1)两个方法声明位置不同sleep()是Thread类中的方法,wait()是Object中的方法
 (2)调用的要求不同,sleep()可以在任何地方调用,而wait()需要在同步代码块或同步方法中调用且调用对象必须是锁对象
 (3)是否释放同步锁的区别,wait()会释放锁,sleep()不会释放锁

**线程通信案例消费者生产者案例**

生产者:
```java
public class ProductDemo implements Runnable {

    private ShareResourceDemo resource;

    public ProductDemo(ShareResourceDemo resource) {
        this.resource = resource;
    }

    @Override
    public void run() {
        for (int i = 0; i <= 50; i++) {
            if (i % 2 == 0) {
                resource.push("tom", "man");
            } else {
                resource.push("lucy", "woman");
            }
        }
    }
}
```
消费者:

```java
public class ConsumerDemo implements Runnable {

    private ShareResourceDemo resource;

    public ConsumerDemo(ShareResourceDemo resource) {
        this.resource = resource;
    }

    @Override
    public void run() {
        for (int i = 0; i <= 50; i++) {
            resource.pop();
        }
    }

}
```
共享资源类:

```java
public class ShareResourceDemo {

    private String name;

    private String sex;

    // 标识共享资源是否为空,true为空 false不为空
    private boolean isEmpty = true;

    /**
     * 生产方法
     *
     * @param name
     * @param sex
     */
    public synchronized void push(String name, String sex) {
        try {
            while (!isEmpty) {
                this.wait();
            }
            // 生产开始
            this.name = name;
            Thread.sleep(10);
            this.sex = sex;
            //生产结束
            isEmpty = false;
            this.notifyAll();
        } catch (InterruptedException e) {
            e.printStackTrace();
        }
    }

    /**
     * 消费方法
     */
    public synchronized void pop() {
        try {
            while (isEmpty) {
                this.wait();
            }
            // 消费开始
            Thread.sleep(10);
            System.out.println(this.name + "--" + this.sex);
            // 消费结束
            isEmpty = true;
            this.notifyAll();
        } catch (InterruptedException e) {
            e.printStackTrace();
        }

    }
}
```
